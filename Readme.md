🛡️ Análise de Vulnerabilidades: OWASP Top 10
📖 Descrição do Projeto
Este projeto foi desenvolvido como parte de um desafio prático no curso de Inteligência Artificial Da DIO Com a RIACHUELLO. O objetivo principal foi utilizar ferramentas de IA Generativa, especificamente o NotebookLM, para sintetizar e explicar de maneira didática os conceitos fundamentais do OWASP Top 10.

O conteúdo foca em facilitar a compreensão das dez vulnerabilidades de segurança de aplicações web mais críticas, auxiliando no processo de aprendizado sobre segurança ofensiva e defensiva.
